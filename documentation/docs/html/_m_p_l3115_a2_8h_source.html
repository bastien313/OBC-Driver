1/*
2 MPL3115A2 Barometric Pressure and Tempurature Sensor Library
3 By: Michael Lange
4 Date: March 31, 2014
5 License: This code is public domain.
6
7 This class wraps the functions of the MPL3115A2 sensor into
8 a usable class that exposes most functions to your project.
9 Functions that are not exposed can easily be added using the
10 existing functions in this library. Specifically, I did not
11 add any funtions to handle FIFO logging or the use of either
12 of the pin interrupts. This should not be too difficult to
13 add if you need those features.
14
15 The motivation here was to get a set of support classes
16 together that supported the chip and could be expanded on.
17 With this library you can extract all relevant data from the
18 sensor.
19
20 Be sure to download the DATASHEET and the App Note AN4519.
21
22 This library was created using the mbed NXP LPC11U24. Pins
23 p27 and p28 were used for the I2C functions. Be sure to install
24 1K pull-up resistors on both lines. Also, if you're not using
25 the SparkFun breakout board, be sure to use the right caps on
26 the power pin. If you don't, the jitter can cause problems.
27
28 This library was inspired by the similar library available for
29 the Arduino written by Nathan Seidle at SparkFun. I copied
30 some of the number crunching routines and tried to follow his
31 style of coding for the library. That way users of Arduinos
32 could step into this library a little easier.
33
34 */
