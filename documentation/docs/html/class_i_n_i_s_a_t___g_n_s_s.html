void INISAT_GNSS::setNMEAoutputChannel ( uint8_t *  NMEAidentifier,
ouputChannel  channel 
)

Set/Get message rate configuration (s) to/from the receiver.

Send rate is relative to the event a message is registered on. For example, if the rate of a navigation message is set to 2, the message is sent every second navigation solution.

Parameters
NMEAidentifierC-string event to be received. Ex: "GGA" or "GSA"
channelChannel were received Channel where the data will be sent. DDC(I2C), USART1, USART2, USB
